Trash Game

2024

A sensoria (Arduino) project, where different sensores like pressure plates and proximity sensores are placed in a real room. Everytime a sensor is triggered it sends a signal to an Wifi-Arduino, which is connected to the machine, running the Trash Game, translating the signal into input, which spawns trash.
